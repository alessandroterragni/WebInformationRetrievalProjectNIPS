Analysis

For the group project assignment, we used a real‐world dataset https://www.kaggle.com/benhamner/nips‐papers as the main information.

This data set includes the title, authors, abstracts, and extracted text for all NIPS papers from 1987 to 2016.

The data source comes in both csv and sqlite files.

NIPS is a flagship machine learning conference and it leads the research trend of machine learning.

In this website the user can search using different parameters, and the information is retrieved using sophisticated algorithms.
